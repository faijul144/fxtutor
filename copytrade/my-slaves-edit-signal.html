The above list includes all the major pairs that we expect you will need to disable. If you need to disable a pair that is not included in the above list you can add it into the box below and click update. This will then add the pair to the list above allowing you to disable it.

Currency Mapping

Sometimes brokers name currency pairs or commodities different names. For example there is a chance that when trading gold, one of our master accounts might trade with the name XAUUSD while your broker names it GLD. In this case, you need to map XAUUSD > GLD so that the system knows that when the master account trades XAUUSD that it should also be trading GLD on your account. Learn more here.

Be aware that this is case sensitive so copy the mapping EXACTLY as your broker has it.
